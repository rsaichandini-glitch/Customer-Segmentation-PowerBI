# Customer Segmentation Analysis using Power BI

## Project Overview

This project focuses on customer segmentation based on income and spending behavior. The objective is to identify different customer groups and provide actionable business insights for targeted marketing strategies.

## Tools Used

- Power BI
- DAX
- Data Visualization

## Dataset

Mall Customers Dataset containing:
- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## Dashboard Features

### KPI Cards
- Total Customers
- Average Income
- Average Spending Score

### Visualizations
- Pie Chart for Customer Segment Distribution
- Scatter Plot for Income vs Spending Analysis
- Bar Chart for Segment Comparison
- Interactive Slicers

## Customer Segments

- VIP Customers
- Potential Customers
- Loyal Customers
- Regular Customers

## Business Insights

- VIP customers generate high revenue.
- Potential customers can be targeted with personalized campaigns.
- Loyal customers show strong engagement.
- Regular customers contribute steady business.

## Outcome

This project demonstrates customer analytics, segmentation techniques, dashboard development, and business intelligence reporting using Power BI.
